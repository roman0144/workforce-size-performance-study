# workforce-size-performance-study
Исследование связи между численностью сотрудников и эффективностью организаций. Проанализированы реальные данные по НКО, исследовательским группам и полиции. Проверено наличие скрытых нелинейных зависимостей. В большинстве случаев масштаб не является предиктором производительности, различия проявляются лишь в крайних значениях.
